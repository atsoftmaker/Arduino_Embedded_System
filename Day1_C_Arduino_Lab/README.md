# C Arduino Lab

ยินดีต้อนรับสู่ Repository **C Arduino Lab** โดย **AtSoftMaker** 🚀

Repository นี้รวบรวมแบบฝึกหัด (Lab Exercises) สำหรับการเขียนโปรแกรมภาษา C/C++ บนแพลตฟอร์ม Arduino และระบบสมองกลฝังตัว (Embedded Systems) ซึ่งออกแบบมาเพื่อใช้ประกอบการอบรมเชิงปฏิบัติการและการพัฒนาทักษะ ตั้งแต่การปูพื้นฐานไปจนถึงการประยุกต์ใช้งานจริง

## 📂 โครงสร้างของ Repository

ภายในโปรเจกต์นี้ประกอบไปด้วยเอกสารคู่มือและซอร์สโค้ดแบบฝึกหัดจำนวน 13 บทเรียน ดังนี้:

* 📄 **`C_Lab_Atsoftmaker.pdf`**: เอกสารคู่มือประกอบการทำ Lab และทฤษฎีที่เกี่ยวข้อง (Lab Manual)
* 💻 **`exercise01/exercise01.ino` - `exercise13/exercise13.ino`**: ซอร์สโค้ดในรูปแบบโครงการย่อยของ Arduino (.ino) แยกตามบทเรียน เพื่อให้เปิดใช้งานบน Arduino IDE ได้โดยตรงและง่ายต่อการเรียนรู้ทีละขั้นตอน

## 🛠️ เครื่องมือที่ต้องใช้ (Prerequisites)

เพื่อให้การทำแบบฝึกหัดเป็นไปอย่างราบรื่น ผู้เรียนควรเตรียมความพร้อมดังนี้:
* โปรแกรม [Arduino IDE](https://www.arduino.cc/en/software) หรือเครื่องมืออย่าง PlatformIO สำหรับเขียนและคอมไพล์โค้ด
* บอร์ดไมโครคอนโทรลเลอร์ (เช่น Arduino UNO, Mega, หรือ ESP32)
* อุปกรณ์ฮาร์ดแวร์และเซนเซอร์พื้นฐานตามที่ระบุในเอกสารคู่มือ (`C_Lab_Atsoftmaker.pdf`)

## 🚀 วิธีการใช้งาน

1. โคลน (Clone) Repository นี้ลงในเครื่องคอมพิวเตอร์ของคุณ:
   ```bash
   git clone [https://github.com/gatunyus/C_Arduino_Lab.git](https://github.com/gatunyus/C_Arduino_Lab.git)
   ```

2. เปิดไฟล์ C_Lab_Atsoftmaker.pdf เพื่ออ่านคำแนะนำ ทำความเข้าใจวงจร และดูโจทย์ของแบบฝึกหัด

3. เปิดโฟลเดอร์ของบทเรียนที่ต้องการ และเปิดไฟล์ `exerciseXX.ino` ด้วยโปรแกรม Arduino IDE ได้ทันที

4. เลือกบอร์ดและพอร์ตให้ถูกต้อง จากนั้นกด Upload เพื่อทดสอบการทำงานบนฮาร์ดแวร์จริง
